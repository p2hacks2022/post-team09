# P2HACKS2022 アピールシート 
### **プロダクトの種類**  
ゲーム

### **タイトル**  
かえりみち

### **コンセプト**  
ユーザーがノスタルジックな世界観を堪能できるゲーム

### **概要**  
【ジャンル】アドベンチャー  
【エンディング数】３  

このゲームはフィクションです。
実在の人物、団体、事件とは一切関係ありません。
また、犯罪や殺生を助長する意図はありません。

### **対象ユーザ**  
若者

### **操作方法**  
>wで上  
>sで下  
>aで左  
>dで右  
  
>fで拾う  
>クリックで次に進む  

![あそびかた](./howToPlay.jpg)

### **推しポイント**  
ビジュアルや世界観にこだわった。
